Ultilize a Tag Header

Usando a Tag header

O heder é ultilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos da tag h1 a h6, até elementos para representar imagens, parágrafos ou mesmo listas de navegação.

Ultilize a Tag article

Usando a Tag Article

Ultilizamos o elemento article quando precisamos declarar um conteúdo que não precisa de otro para fazer sentido em um documento em HTML, por exemplo, um artigo em um blog. É recomendado identificar cada aricle com um título.

Utilize a Tag section

Usando o Section

Seção 1

O elemento section representa uma seção dentro dentro do documento e geralmente contém um título, o quel é definido por meio de um dos elementos entre h1 e h6. Podemos utilizar o section, por exemplo, para descrever as seções/tópicos de um documento.

Ultilize a tag div
